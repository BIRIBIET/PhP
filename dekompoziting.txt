-Как создать Блог;
1Э
Что такое блог?
1)
- Это сайт
Что такое Сайт?  - если это сайт то у него должна быть главная сраница.
Сделать главную страницу.
- загаловок статьи(h3)
- краткий текст(р)
- ссылка на пост(а)
2)
- Размещение информации..
- информацию могут добовлять люди у которых нет знания о программирования..
- Нужно создать отдельнюу страницу для Админ панеля..
- Создать базу данных..
3)
- Посты
Что такое Посты? - это статья созданное пользователя сайта.  
- Загаловок..
- Кратное опмсание..
- Подробное описание статьи..
- Сделать отдельную страницу для поста..

4)База данных
- Создать базу данных
- тип пер. обьект
- Сохранение цифровых, текстовых и изображенных файлов
ключи:
- загаловок("main")
- краткое писание("short_text")
- подробное описание ("long_text")

Шаги:
1. Создать Главную страницу
2. Создать Базу Данных
3. Создать отдельную страницу для Админ панели
4. Сделать отдельную страницу для поста
